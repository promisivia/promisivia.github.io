Projects

Storage System with Non-volatile Memory (NVM)

2021.06 - 2023.10

Non-volatile Memory (NVM) combines the high performance of memory devices with the persistence of storage devices. This project focuses on designing new storage systems and operating systems based on NVM to improve system performance and data persistence. Key achievements include:

(1) HTMFS (FAST 22): The first file system leveraging NVM eADR hardware features to achieve low-overhead high-consistency file system.

(2) TreeSLS (SOSP 23): The first whole-system persistent operating system based on NVM, replacing the traditional memory/disk two-layer storage structure with a single-layer storage structure.

System Research Based on New Interconnection Bus

2024.01 - Present

New interconnection buses (such as CXL) enable direct memory access between multiple machines. This project focuses on leveraging new interconnection bus capabilities to design new underlying synchronization mechanisms, system critical components (such as file systems, object storage services), and build unified distributed operating systems to improve application performance and overall cluster resource utilization.
